\problem{57-I-1-D2}{}{
Pre koľko usporiadaných trojíc prirodzených čísel $x, y, z$ platí $xyz = 1 000 000$?
}{
\rieh Návod: $1 000 000 = 2^6\cdot5^6$. Položme $x = 2^a5^p$, $y = 2^b5^q$, $z = 2^c5^r$ a preskúmajme všetky možnosti pre $a + b + c = 6$ a pre $p + q + r = 6$. Nakoniec zistíme hľadaný počet: $28\cdot28 = 784$.
}

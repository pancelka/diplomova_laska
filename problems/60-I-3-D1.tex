\problem{60-I-3-D1}{

Obsahy trojuholníkov $ADP$, $DCP$, $DBQ$, $CDQ$ označme postupne $S_1$, $S_2$, $S_3$, $S_4$. Vypočítajte $S_1 : S_3$, ak $S_1 : S_2 = 2 : 3$ a $S_3 : S_4 = 3 : 8$.
}{
